# Python Banking Management System

## Description

A simple command-line banking application developed in Python. Users can check balance, deposit money, and withdraw money.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Simple and user-friendly interface

## Technologies Used

- Python 3

## How to Run

1. Download the project
2. Open terminal
3. Run:

```bash
python banking_system.py
```
